# ギャラリー画像フォルダ

このフォルダに画像ファイルを入れるだけで、自動的にサイトのAboutセクションのギャラリーに表示されます。

## 使い方

1. このフォルダ（`assets/local-images/gallery/`）に画像ファイルをコピーしてください
2. ファイル名は自由です（例：`photo1.jpg`, `馬の写真.png` など）
3. 対応している形式：`.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`, `.avif`
4. 画像はファイル名の順番で自動的に並びます

## 表示場所

- Aboutセクション（ABOUT）の下部にあるギャラリーエリアに表示されます
- 画像を追加・削除すると、自動的にサイトに反映されます（開発サーバーの再起動が必要な場合があります）

## 注意事項

- このフォルダに画像が入っている場合、個別指定（`about-gallery-1`など）は無視されます
- このフォルダが空の場合のみ、個別指定の画像が使用されます
- 推奨サイズ：正方形（1:1）または縦長（4:3）の画像
