# ロゴ画像フォルダ

このフォルダにロゴ画像ファイルを入れると、サイト全体で使用されるロゴとして自動的に表示されます。

## 表示場所

- ヘッダー（サイト上部）の左上
- Heroセクション（トップページ）の中央
- フッター（サイト下部）の左側

## 使い方

1. このフォルダにロゴ画像ファイルをコピーしてください
2. ファイル名は自由です（例：`logo.png`, `kazu-logo.svg` など）
3. 対応している形式：`.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`, `.avif`
4. 最初に見つかった画像が使用されます

## 推奨サイズ

- 正方形または横長のロゴ
- 幅：300px〜500px推奨
- 背景が透明なPNGまたはSVG形式を推奨
