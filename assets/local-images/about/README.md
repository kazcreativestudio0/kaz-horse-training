# About セクション画像フォルダ

このフォルダに画像ファイルを入れると、Aboutセクション（ABOUT）のメイン画像として自動的に表示されます。

## 表示場所

- Aboutセクションの左側メイン画像

## 使い方

1. このフォルダに画像ファイルをコピーしてください
2. ファイル名は自由です（例：`about.jpg`, `main-photo.png` など）
3. 対応している形式：`.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`, `.avif`
4. 最初に見つかった画像が使用されます

## 推奨サイズ

- 縦長の画像（3:4または2:3）
- 幅：800px〜1200px推奨
