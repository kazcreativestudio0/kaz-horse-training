# 地図画像フォルダ

このフォルダに画像ファイルを入れると、Contactセクション（アクセス）の地図エリアに表示されます。

## 表示場所

- Contactセクションの地図エリア

## 使い方

1. このフォルダに地図画像ファイルをコピーしてください
2. ファイル名は自由です（例：`map.jpg`, `access-map.png` など）
3. 対応している形式：`.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`, `.avif`
4. 最初に見つかった画像が使用されます

## 推奨サイズ

- 横長の画像（16:9または4:3）
- 幅：800px〜1200px推奨
