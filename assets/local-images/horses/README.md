# 馬の紹介画像フォルダ

このフォルダに画像ファイルを入れると、Horsesセクション（馬の紹介）で使用されます。

## 表示場所

- メイン3頭の紹介カード
- 追加ギャラリー

## 使い方

1. このフォルダに画像ファイルをコピーしてください
2. ファイル名は自由です（例：`horse1.jpg`, `ウエスタンホース.png` など）
3. 対応している形式：`.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`, `.avif`
4. 画像はファイル名順に自動的に並びます
5. 最初の3枚がメイン3頭として使用され、残りがギャラリーに表示されます

## 推奨サイズ

- 横長の画像（4:3）
- 幅：800px〜1200px推奨
