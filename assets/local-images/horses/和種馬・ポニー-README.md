このフォルダには、**和種馬・ポニー**（道産子・木曽馬・ポニー）の画像を配置します。

## 📋 カテゴリ詳細

### 特徴
- **種**: 道産子・木曽馬・ポニー
- **特徴**: 日本の在来馬
- **性格**: 小柄で我慢強い
- **対象**: 子供たちの良きパートナー、小さなお子様や初心者に最適

### 詳細情報

**日本の在来馬**
北海道和種（道産子）や木曽馬、ポニーなど、日本の伝統的な馬種も在籍した実績があります。これらの在来馬は、日本の風土に適応してきた丈夫で我慢強い性格を持っています。小柄で穏やかな性格のため、特に小さなお子様や初心者の方に最適です。

**小柄で我慢強い**
小柄な体格でありながらも、我慢強く優しい性格の馬たちは、子供たちの良きパートナーとして最適です。2歳のお子様でも、やる気があれば引き馬体験からスタートできます。スタッフが常にサポートしますので、安心してご体験いただけます。

**ファミリーに優しい**
家族での乗馬体験にも最適です。親子での引き馬体験や、お子様の体験乗馬を見守りながら保護者の方もレッスンを受けていただけます。小さな頃から馬と触れ合うことで、思いやりや責任感が育まれます。

### 体験できるプログラム
- ✓ 引き馬体験（2歳のお子様から可能）
- ✓ 家族での乗馬体験
- ✓ お子様向けレッスン（9歳以上推奨）
- ✓ 馬とのふれあい体験

## 🖼️ 画像の使い方

### 推奨される画像
- 小さなお子様と馬のふれあいシーン
- 引き馬体験の様子
- 家族で楽しむ様子
- 小柄で優しい表情の馬の写真
- ポニーとの触れあい

### ファイル名
- `horse-1.jpg`, `horse-2.jpg` など（ファイル名順に使用されます）
- 最初の3枚がメイン3頭のカードに表示されます
- 4枚目以降は追加ギャラリーに表示されます

### 推奨サイズ
- 横長の画像（4:3）
- 幅：800px〜1200px推奨
- 対応形式：`.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`, `.avif`

## 📝 備考

このカテゴリの画像は、サイトの「馬の紹介」セクションで使用されます。各画像はクリックすると詳細ページが開きます。小さなお子様との触れあいが映える写真を選ぶと、ファミリー層への訴求力が高まります。
