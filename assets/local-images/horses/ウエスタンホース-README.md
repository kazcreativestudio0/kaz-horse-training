# ウエスタンホース - 画像フォルダ

このフォルダには、**ウエスタンホース**（クォーターホース種など）の画像を配置します。

## 📋 カテゴリ詳細

### 特徴
- **種**: クォーターホース種など
- **調教**: ウエスタン競技で活躍できるよう丁寧に調教
- **性格**: とても穏やかで人懐っこい
- **対象**: 初心者の方でも安心して騎乗可能

### 詳細情報

**ウエスタン競技対応**
カズホーストレーニングのウエスタンホースたちは、ウエスタン競技で活躍できるよう丁寧に調教されています。クォーターホース種を中心とした馬たちは、スピードと機動性に優れ、ウエスタン競技の特性を活かしたトレーニングを受けています。

**穏やかで人懐っこい性格**
とても穏やかで人懐っこく、初心者の方でも安心して騎乗できます。馬の心理を重視したナチュラルホースマンシップ（馬の自然な習性に沿った調教法）を実践し、馬にストレスをかけない丁寧なトレーニングを行っています。人と馬が一対一で向き合う楽しさを感じていただけます。

**ウエスタンスタイル**
ウエスタン乗馬は、カウボーイ文化にルーツを持つカジュアルなスタイルです。ウエスタンサドルは大きく安定感があるため、初めての方でも安心感があります。デニムやブーツなど動きやすい服装で気軽に体験できるのが魅力です。

### 体験できるプログラム
- ✓ 体験乗馬15分コース（自分で手綱を持って操作）
- ✓ 体験乗馬45分コース（速歩や軽い駈歩まで挑戦）
- ✓ 本格的なウエスタン乗馬レッスン
- ✓ 馬のブラッシング体験

## 🖼️ 画像の使い方

### 推奨される画像
- ウエスタンハットをかぶったライダーと馬
- ウエスタンサドルを装着した馬
- 競技やトレーニング中の様子
- 穏やかな表情の馬の写真
- 初心者とのふれあいシーン

### ファイル名
- `horse-1.jpg`, `horse-2.jpg` など（ファイル名順に使用されます）
- 最初の3枚がメイン3頭のカードに表示されます
- 4枚目以降は追加ギャラリーに表示されます

### 推奨サイズ
- 横長の画像（4:3）
- 幅：800px〜1200px推奨
- 対応形式：`.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`, `.avif`

## 📝 備考

このカテゴリの画像は、サイトの「馬の紹介」セクションで使用されます。各画像はクリックすると詳細ページが開きます。
