# 仔馬たち - 画像フォルダ

このフォルダには、**仔馬たち**（生産馬・繁殖育成）の画像を配置します。

## 📋 カテゴリ詳細

### 特徴
- **種**: 生産馬
- **特徴**: 繁殖・育成プログラム
- **性格**: 元気いっぱい
- **対象**: 生命の成長を間近で感じたい方

### 詳細情報

**繁殖・育成プログラム**
当クラブでは繁殖・育成も行っています。若馬の初期訓練から問題行動の矯正まで対応し、プロの手で調教・訓練するサービスも提供しています。仔馬の成長過程を見守り、ともに育つ喜びを味わっていただけます。

**元気いっぱいの仔馬たち**
時期によっては、元気いっぱいに放牧場を駆け回る仔馬たちの姿を見ることができるかもしれません。自由に走り回る姿は、見ているだけでも癒やされます。生命の力強さと成長の過程を間近で感じることができる、特別な体験です。

**ホースマンシップクリニック**
馬の行動心理や調教方法を学ぶクリニックも不定期で開催しています。馬への接し方やナチュラルホースマンシップの理論を、実演やワークショップ形式で体験できます。馬をもっと理解したい、コミュニケーションを極めたい方向けの内容です。

### 体験できるプログラム
- ✓ 仔馬とのふれあい（時期により）
- ✓ 繁殖・育成の見学
- ✓ ホースマンシップクリニック
- ✓ 調教預託サービス（ご相談）

## 🖼️ 画像の使い方

### 推奨される画像
- 放牧場で走る仔馬の姿
- 仔馬とのふれあいシーン
- 母馬と仔馬の様子
- 育成・トレーニングの様子
- 元気いっぱいの表情の写真

### ファイル名
- `horse-1.jpg`, `horse-2.jpg` など（ファイル名順に使用されます）
- 最初の3枚がメイン3頭のカードに表示されます
- 4枚目以降は追加ギャラリーに表示されます

### 推奨サイズ
- 横長の画像（4:3）
- 幅：800px〜1200px推奨
- 対応形式：`.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`, `.avif`

## 📝 備考

このカテゴリの画像は、サイトの「馬の紹介」セクションで使用されます。各画像はクリックすると詳細ページが開きます。生命の力強さや成長の過程が感じられる写真を選ぶと、訪れた方に感動を与えます。

**注意**: 仔馬の様子は季節や時期によって異なります。最新の状況については、お電話でお問い合わせください。
