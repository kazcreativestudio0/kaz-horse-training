# 料金プラン画像フォルダ

このフォルダに画像ファイルを入れると、料金プランセクションの各プランカードに表示されます。

## 表示場所

- 体験乗馬（ショート）プラン
- 体験乗馬（ロング）プラン
- レッスン回数券プラン

## 使い方

1. このフォルダに画像ファイルをコピーしてください
2. ファイル名に以下を含めると、自動的に該当プランに割り当てられます：
   - `trial` を含むファイル → 体験乗馬（ショート）
   - `horse-care` または `horsecare` を含むファイル → 体験乗馬（ロング）
   - `lesson` を含むファイル → レッスン回数券
3. 対応している形式：`.png`, `.jpg`, `.jpeg`, `.webp`, `.svg`, `.avif`
4. 該当するファイル名がない場合、最初に見つかった画像が使用されます

## 推奨サイズ

- 横長の画像（16:9または4:3）
- 幅：800px〜1200px推奨
