# カズホーストレーニングWEBサイト - 画像配置一覧

## ローカル画像フォルダの使い方

- フォルダ: `assets/local-images/`
- 対応しているファイル形式: `.png`, `.jpg`, `.jpeg`, `.webp`
- ファイル名がそのままキーになります（拡張子を除いた部分を使用）
- 例: `hero.jpg` を配置すると、自動的に Hero 背景として読み込まれます

| ファイル名 | 用途 |
| --- | --- |
| `logo.*` | ロゴ (Header / Hero / Footer) |
| `hero.*` | Hero背景 |
| `about.*` | Aboutメイン画像 |
| `about-gallery-1`〜`about-gallery-7` | Aboutギャラリー1〜7 |
| `horse-1`〜`horse-6` | 馬の紹介およびギャラリー |
| `plan-trial` / `plan-horse-care` / `plan-lesson` | 料金プラン各カード |
| `map.*` | Contactセクションの地図背景 |

※ ファイルを入れ替えるだけで、自動的に `constants.ts` へ反映されます。

## 🔧 画像を変更する方法

### 1. ローカル画像で運用する場合（推奨）
1. `assets/local-images/` フォルダに対応するファイル名で画像を配置
2. ブラウザを更新すると自動で反映
3. ファイル名を変更すれば、同じキーで別画像に差し替え可能

### 2. 外部リンクを使う場合
1. `constants.ts` の `resolveImage('キー', 'fallbackUrl')` の第2引数を書き換え
2. Google Driveの場合は共有リンクを貼るだけで自動変換されます
