# アンチグラビティデザインの戻し方

このファイルは、アンチグラビティデザインを元に戻す手順を説明します。

## ⚠️ 現在のブランチ

現在、`anti-gravity-design` ブランチにいます。
元のデザインに戻すには、以下のいずれかの方法を使用してください。

## 🔄 戻し方

### 方法1: mainブランチに戻る（最も簡単）

```bash
git checkout main
```

これで、アンチグラビティデザインが適用される前の状態に戻ります。

### 方法2: 変更を元に戻す（現在のブランチで）

```bash
# すべての変更を元に戻す
git restore .

# または特定のファイルだけ
git restore index.html
git restore components/Hero.tsx
# ... など
```

### 方法3: ブランチを削除する

```bash
# mainブランチに戻る
git checkout main

# anti-gravity-designブランチを削除
git branch -D anti-gravity-design
```

## ✅ 確認方法

戻した後、以下で確認できます：

```bash
# 現在のブランチを確認
git branch

# 変更がないことを確認
git status
```

## 📝 注意事項

- この変更はまだコミットされていないため、簡単に戻せます
- 気に入った場合は、このブランチでコミット・プッシュできます
- 元のデザインに戻した後、再度アンチグラビティデザインを試したい場合は、再度実装します
